This project is an attempt to allow readers of {% include "patterns/atoms/link/link.html" with text="The Wandering Inn" href="https://wanderinginn.com/" external=True %} web serial to view and search interesting statistics regarding the many [Classes], [Skills], [Spells], characters, and unique items that pop up throughout the story.

It is also my hope, that this will serve as a useful tool for my fellow Wiki Warriors when searching TWI when trying to identify errors in the {% include "patterns/atoms/link/link.html" with text="Wiki" href="https://thewanderinginn.fandom.com/wiki/The_Wandering_Inn_Wiki" external=True %} as well as help finding relevant information characters, new skills, spells, and classes as the story unfolds. Please consider contributing to the {% include "patterns/atoms/link/link.html" with text="repository" href="https://github.com/cblanken/wandering-inn-react" external=True %} by submitting an {% include "patterns/atoms/link/link.html" with text="issue" href="https://github.com/cblanken/wandering-inn-react/issues" external=True %} if you've found any information here to be inaccurate or any other errors.

Please note that this application is not officially endorsed by pirateaba or The Wandering Inn. If you enjoy the web serial, I encourage you to subscribe to pirateba's {% include "patterns/atoms/link/link.html" with text="Patreon" href="https://www.patreon.com/pirateaba" external=True %} or check out the {% include "patterns/atoms/link/link.html" with text="official store" href="https://www.store.wanderinginn.com" external=True %} .

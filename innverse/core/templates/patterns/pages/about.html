About

Welcome my fellow ducks! This project is an attempt to allow readers of {% include "patterns/atoms/link/link.html" with text="The Wandering Inn" href="https://wanderinginn.com/" external=True %} web serial to explore interesting statistics regarding the many [Classes], [Skills], [Spells], characters, locations, and unique items that pop up throughout the story. I also hope this application will serve as a useful tool for all the [Wiki Warriors] when searching TWI for references and identifying errors in the {% include "patterns/atoms/link/link.html" with text="Wiki" href="https://thewanderinginn.fandom.com/wiki/The_Wandering_Inn_Wiki" external=True %}.

If you've found any errors or would like to request any additional features, please consider submitting an {% include "patterns/atoms/link/link.html" with text="issue" href="https://github.com/cblanken/wandering-inn-stats/issues" external=True %}. Thanks!

Please note that this application is not officially endorsed by pirateaba or The Wandering Inn. If you enjoy the web serial, I encourage you to support the author by subscribing to pirateba's {% include "patterns/atoms/link/link.html" with text="Patreon" href="https://www.patreon.com/pirateaba" external=True %} or check out the {% include "patterns/atoms/link/link.html" with text="official store" href="https://www.store.wanderinginn.com" external=True %} .

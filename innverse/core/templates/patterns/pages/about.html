Welcome! This project is an attempt to allow readers of {% include "patterns/atoms/link/link.html" with text="The Wandering Inn" href="https://wanderinginn.com/" external=True %} web serial to view and search interesting statistics regarding the many [Classes], [Skills], [Spells], characters, and unique items that pop up throughout the story.

I also hope this application will serve as a useful tool for my fellow Wiki Warriors when searching TWI to help complete the {% include "patterns/atoms/link/link.html" with text="Wiki" href="https://thewanderinginn.fandom.com/wiki/The_Wandering_Inn_Wiki" external=True %} and identify any errors. as well as help finding relevant references of characters, new skills, spells, and classes as the story unfolds. Please consider submitting an {% include "patterns/atoms/link/link.html" with text="issue" href="https://github.com/cblanken/wandering-inn-react/issues" external=True %} if there are any features you'd like to be considered or if you've found any errors. Thanks!

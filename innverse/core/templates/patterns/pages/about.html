The Project

Welcome my fellow ducks! This project is an attempt to let readers of {% include "patterns/atoms/link/link.html" with text="The Wandering Inn (TWI)" href="https://wanderinginn.com/" external=True %} web serial explore interesting statistics of the many [Classes], [Skills], [Spells], characters, locations, etc. that show up throughout the story.

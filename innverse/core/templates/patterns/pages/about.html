About

The Project

Welcome my fellow ducks! This project is an attempt to allow readers of {% include "patterns/atoms/link/link.html" with text="The Wandering Inn" href="https://wanderinginn.com/" external=True %} web serial to explore interesting statistics regarding the many [Classes], [Skills], [Spells], characters, locations, etc. that show up throughout the story. I also hope this application will serve as a useful tool for all the [Wiki Warriors] when searching TWI for references and identifying errors in the {% include "patterns/atoms/link/link.html" with text="Wiki" href="https://wiki.wanderinginn.com" external=True %}.

If you notice any errors, strokes, or glitches, or even want to request a feature, please consider submitting an {% include "patterns/atoms/link/link.html" with text="issue" href="https://github.com/cblanken/wandering-inn-stats/issues" external=True %}

Thanks :)
